# My Life Dashboard (個人生活儀表板)

這是一個基於 Web (HTML/JS) 與 Google Sheets (GAS) 的個人財務與生活追蹤系統。
整合了記帳、資產管理、習慣打卡與目標倒數功能。

## 主要功能

* **資產戰情室**：即時計算總資產、投資成本、未實現損益與 ROI。
* **生活財務室**：計算每月可用生活資金，並設定 **每月預算上限** 進行監控。
* **趨勢分析**：雙軸圖表顯示「每月支出」vs「資產成長趨勢」。
* **自律熱力圖**：GitHub 風格的年度習慣打卡牆，視覺化你的一整年努力。
* **夢想購物車**：許願想買的清單，系統自動判斷餘額是否足夠購買。
* **時間倒數**：年度進度條、畢業倒數、生日倒數 (支援動態新增)。
* **習慣打卡**：每日追蹤英文、運動、閱讀等習慣，計算複利成長曲線。
* **閱讀紀錄**: 去紀錄自己的閱讀、電影心得。
* **手機友善 (PWA)**：支援加入手機主畫面，全螢幕操作體驗。
* **隱私安全**：採用「智慧登入」系統，網址與密碼僅儲存於您的瀏覽器。

## 如何開始使用 (Setup)

本專案不需要修改程式碼，直接開啟網頁即可設定。

### 第一步：準備後端 (Google Sheet)
1. 建立一個新的 Google Sheet。
2. 依照格式建立分頁 (`Asset_Log`, `Expenses_DB`...等)。
3. 開啟 **擴充功能** -> **Apps Script**，貼上後端程式碼。
4. 點擊 **部署 (Deploy)** -> **新增部署** -> **網頁應用程式** -> 權限設為 **所有人 (Anyone)**。
5. **複製獲得的 GAS 網址**。

### 第二步：啟動儀表板
1. 直接點擊 GitHub Pages 網址開啟網頁。
2. 網頁會跳出視窗，請貼上您剛剛複製的 **GAS 網址**。
3. 設定您的 **專屬登入密碼**。

##  Google Sheet 格式需求

後端對應的 Google Sheet 需包含以下分頁 (Tabs)：
* `Asset_Log` (資產流水帳)
* `Expenses_DB` (日常支出)
* `Portfolio` (資產現況與股價公式)
* `Habit_Log` (習慣紀錄)
* `History_Log` (資產快照存檔)
* `Events_DB` (倒數事件)
* `Wishlist_DB` (購物清單)
* `Media_Log` (閱讀觀影紀錄)

## 常見問題

**Q: 如何修改「每月預算」金額？**
A: 預算金額目前設定在程式碼中。請開啟 `index.html`，搜尋 `const MONTHLY_BUDGET`，修改後面的數字即可。

**Q: 如果我換手機或換電腦怎麼辦？**
A: 在新裝置上打開網頁，系統會再次詢問 GAS 網址與密碼，輸入一次即可記住。

**Q: 如何重設網址或密碼？**
A: 在登入畫面故意輸入錯誤密碼，會出現「重設所有設定」的按鈕。
